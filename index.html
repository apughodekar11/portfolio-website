🔨 Project

gRPC Stream Load Balancer

Application-level gRPC load balancer in Python with health-based ejection, graceful connection draining, and stream failover — transparently re-routing an in-flight client stream to a healthy backend when one dies mid-stream. Replays real OpenSSH Loghub logs as the stream payload. CI via Azure Pipelines (Test -> Build to ACR), driven end-to-end through the Azure DevOps MCP server and GitHub Copilot.
